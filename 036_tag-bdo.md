# HTML &lt;bdo&gt; 标签

## 实例

指定文本方向：

```HTML
<p>该段落文字从左到右显示。</p> 
<p><bdo dir="rtl">该段落文字从右到左显示。</bdo></p>
```

## 标签定义及使用说明

bdo 指的是 bidi 覆盖（Bi-Directional Override）。

&lt;bdo&gt; 标签用来覆盖默认的文本方向。

--------

## HTML 4.01 与 HTML5之间的差异

无。

| ---- | ---- | ---- |
