# HTML &lt;sub&gt; 标签

## 实例

下标文本：

```HTML
<p>这个文本包含 <sub>下标</sub>文本。</p>
```

## 标签定义及使用说明

&lt;sub&gt; 标签定义下标文本。下标文本将会显示在当前文本流中字符高度的一半为基准线的下方，但是与当前文本流中文字的字体和字号都是一样的。下标文本能用来表示化学公式，比如 H2O。

**提示：** 请使用 [&lt;sup&gt;](121_tag-sup.md) 标签定义上标文本。
